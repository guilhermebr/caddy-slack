# slack

Plugin for [Caddy](https://caddyserver.com).

**slack** integrate caddy and slack.

### Syntax

```
slack [hook_token]
```
* **hook_token** is the slack incoming_webhook (https://my.slack.com/services/new/incoming-webhook/)

## Features

This first version only send caddy logs for a slack team/channel.

Next versions will integrate with slack bots. Sugestions please open a issue.
